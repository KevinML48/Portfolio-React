Prototypage Application

Dans le cadre du projet "Jardinons Ensemble" avec le Jardin collectif de Vaugelas à Villefontaine, j’ai développé une application en cinq parties pour accompagner les jardiniers. La première présente le jardin et les jardiniers. La seconde offre des ressources vidéo pour apprendre des techniques de jardinage. La troisième est un calendrier pour planifier les tâches du jardin. La quatrième est un plan pour se repérer dans le jardin, et la dernière est le profil de chaque jardinier, incluant leurs compétences et permettant la discussion entre eux.
